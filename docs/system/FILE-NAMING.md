# FILE-NAMING

- User agent: `daily-<userId>`
- User workspace: `workspace-daily-<userId>`
- Task file: `TASK.json`
- Plan file: `PLAN.md`
- Decisions: `DECISIONS.md`
- Acceptance: `ACCEPTANCE.md`
- Next steps: `NEXT_STEPS.md`
- Signoff: `SIGNOFF.json`
- Daily memory: `memory/YYYY-MM-DD.md`
- Long-term memory: `MEMORY.md`
- User index: `state/users/index.json`
- Agent catalog: `state/agents/catalog.json`
- Skill catalog: `state/skills/catalog.json`
- Task state: `state/router/tasks.json`
- Workflow truth: `workflows/*.json`
- Policies: `policies/*.json`
