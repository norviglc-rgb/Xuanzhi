- `SOUL.md`
- `IDENTITY.md`

- `BOOT.md`
- `HEARTBEAT.md`

## Constraint Model

- Hard runtime boundaries live in `openclaw.json` and managed hooks.
- Shared machine-readable governance lives in `~/.openclaw/skills/xuanzhi-control/`.
- Workspace markdown guides behavior but does not expand permissions.
